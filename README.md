# Android AI Diet Consultant System

Simple frontend assignment project using pure HTML, CSS, and JavaScript.

## Structure

- `index.html`: app shell and script loading
- `styles/`: responsive card-based layout
- `components/`: reusable UI render helpers
- `models/`: app state and mock data
- `controllers/`: login, BMI, calories, diet plan, feedback logic
- `views/`: screen templates only
- `main.js`: navigation and event binding

## How to run

Open `index.html` directly in a browser. No backend, npm, framework, or build step is required.
